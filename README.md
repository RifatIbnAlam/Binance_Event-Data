# Binance_Event-Data

A dataset repository containing Binance cryptocurrency trading data organized around key market events for event study analysis.

## Overview

This repository provides structured cryptocurrency price and trading data from Binance, organized by specific event dates (March, June, and November). The data is intended for use in financial event studies and market reaction analyses.

## Repository Contents

- `March_Data.csv` — Binance trading data around the March event
- - `June_Data.csv` — Binance trading data around the June event
  - - `November_Data.csv` — Binance trading data around the November event
    - - `March/` — Detailed sub-data for the March event period
      - - `June/` — Detailed sub-data for the June event period
        - - `November/` — Detailed sub-data for the November event period
         
          - ## Use Cases
         
          - - Cryptocurrency market event studies
            - - Abnormal return analysis around specific dates
              - - Financial data analysis and modeling
               
                - ## Tools Used
               
                - - Python / Jupyter Notebook
                  - - Pandas, NumPy
